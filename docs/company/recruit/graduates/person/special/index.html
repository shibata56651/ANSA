TALK 01オリコとの出会いや、面接時の印象、そして入社の決め手となったことは？

大場
大学は経済学部だったのですが、ゼミでキャッシュレスの研究をしていました。そのゼミ活動の中で金融に興味を持ったので、金融業界に軸を置いて就活をしていました。いくつかの会社を知っていく中で、特に社風が良く感じたオリコを選びました。
貞木
私も社風がオリコを選んだ決め手です。私は業界を絞らず就活をしていましたが、風通しの良さそうな社風に惹かれました。「かなえる、のそばに。」というブランドスローガンに共感したことも決め手のひとつですね。
大場
わかります。僕も「かなえる、のそばに。」を見つけた時に直感的に「いいな」と感じました。
貞木
面接の時も、とても気さくな方が面接官で、親身に話を聞いてくださいました。こちらが気になった点についても、ご自身の経験からお話をしてくださったので、イメージが湧きやすかったですね。そういったところからも、風通しや環境が良さそうという印象がありました。カジュアルな言い方だと、アットホームな雰囲気といった感じでしょうか。
私自身、今までの人生では、とても人に恵まれていたと思っています。多くの人に支えられて成長してきたので、漠然と社会に貢献できるような仕事がしたいと思っていました。オリコの「かなえる、のそばに。」という言葉に共感し、私も誰かの人生をサポートしたいと思いました。それが入社の決め手になりました。
増田
私は、軸を将来性と安定性の2点を軸に就活を行っていました。就活がコロナの時期だったことも、この2つを重要視した要因だと思います。ちなみに初めはキャッシュレス業界を見ていて、そのなかでオリコを知りました。オリコは信販業界の最大手ですし、その安定性と将来性もまさに理想的でした。
また、面接をしてくれた社員の印象が良かったことも入社を決めた要因の一つです。常に聞き手に回っていただき、私が自然にアピールできるような環境を作ってくださったのかなと思いますが、柔らかな印象で自分の思いをしっかりと伝えることができました。
貞木
私も面接の時に面接官が「人事部は社員のために仕事をしている」とおっしゃっていたのを今、思い出しました。その時に「オリコは社員を大切にする会社なんだな」と感じて、そこに惹かれました。
ちなみにその方は、当時の人事部長さんでした。内定式で「見たことがある方だな」と思っていたら人事部長であることが分かり、驚いたのを覚えています。（笑）

TALK 02各部署に配属されてほぼ2ヶ月ですが、実際の仕事はどうでしょうか？

大場
僕は、さいたま支店で、ショッピングクレジットやリフォームローンの利用促進の営業推進活動を行っています。
僕が今一番感じているのは、「先輩方から手厚くフォローしていただける」ということです。6ヵ月間の研修を終えて今は営業として2ヵ月ほど経過したのですが、研修の間は教育チームの方に何でも聞ける環境が整っていました。何でもすぐに質問していたのですが、トレーナーの方は手を止めて丁寧に教えてくださいました。研修が終わり、2ヵ月経った今でも「分からないことがあったら何でも聞いて」と言ってもらえます。すごくアットホームというか、新入社員に対してとても優しい会社だなと思いますね。
貞木
私は、債権管理業務で、ご契約中のお客さまの入金管理業務を担当しています。勤務地は、千葉管理センターというところです。ご事情があってお支払いが遅れてしまうお客さまもいらっしゃいますので、こちらから電話をしてお支払いのご相談をしています。
この仕事では、1日のうちで電話をかける時間帯や回数がルールで決められています。コンプライアンス研修を受けた後は、実際の業務を通じて仕事を学ぶことになります。
一見大変そうですが、一件一件、お客さまのご事情をお伺いしながら、お客さまに寄り添った対応をしています。お客さまに最適なご提案ができた時に、大きなやりがいを感じています。
増田
私は、営業事務を担当しています。勤務地は東京本社です。各種ローンの申込受付や残金の一括処理、ローンを申込まれたお客さまに契約内容の確認電話をしたりする業務を主に行っています。私が担当している業務は、月末に申込みが集中する傾向があります。月末は大変忙しくなるのですが、ローンのお申込みから契約内容の確認電話をするまでの時間を短くすると、お客さまも加盟店さまも喜んでくださいます。CS（顧客満足度）を向上させ、チームメンバーと協力しながら月末を乗り切った時には、大きな達成感がありますね。

TALK 03実際に仕事を始めてみて、見えてきたものはありますか？ 新しく始めた行動は？

貞木
先ほど増田さんがCSについてお話しされていましたが、CSに関しては私の業務でも意識しています。お客さまと直接お話する時、相手側の満足度に対する意識を高く持っていますし、実際の業務を通じてその技術を磨いているところです。
お客さまの安心につながるクッション言葉や言葉使いができているか、電話のモニタリングなども行って、日々スキル向上を図っています。お客さま対応スキルを磨いて、自身の成長につなげていきたいと思っています。
将来的には、身につけたスキルを活かしどのような難しい案件を担当したとしても、お客さまに「オリコを利用してよかった」と思っていただけるようサービスを提供していきたいです。
大場
僕は、「まず営業として一人前になりたい」という思いが強いです。その後は、海外事業部に行きたいという希望を出していています。年に何回か支店長や人事の方にキャリアについて話を聞いてもらえる機会があるので、その場ではしっかりと自身が思い描くキャリアを伝えています。
増田
私も、まずは今いる部署で一人前になること、チームメンバーから頼られる人になることが最優先です。今の業務では、与えられたことをすることが多いのですが、自分から発信できるような人になっていきたいですね。
もともとキャッシュレス事業に携わりたいと思ってオリコに入社したので、何年か後にはなるかもしれませんが、そういったことを推進できるような仕事をしたいなと思います。
実は、今後さまざまなことに携われるように、FPの勉強もしています。少しでも多くの資格をとって、いつか活かせるようになりたいですね。

TALK 04最後に就活中の皆さんへの一言をお願いします。

大場
自分のやりたいことをしっかりと考えながら会社を見ていけば、就職活動は上手くいくのではないかと思います。頑張ってください。
貞木
私は、やるべきことをやれば自ずと結果は付いてくると思います。だからこそ、「やるところはキッチリやる」ということを意識してみてほしいです。
就活生の方の中には、心配事を抱えている方も多いと思います。ただ、あまりそこを心配しすぎずに、上手に肩の力を抜いてリフレッシュしてほしいです。基本的には自分のペースで進めながら、「やるべきところはキッチリ」とメリハリをつけて就活すれば良いのではないかなと思います。
増田
大学生の時というのは、さまざまな会社を見られる最後のチャンスだと思います。だからこそ、早い時期から1社に絞らずいろいろな会社を見たり、自分が興味のあることに1つでも多く挑戦したりしてみてください。それが、やりたいことが見つかるきっかけになるかもしれません。ぜひ、楽しみながらやってみてほしいですね。
貞木
私はあまり根を詰めないタイプなのですが、リフレッシュしたい時には人に話を聞いてもらっていました。根を詰めるタイプの友達を見て「大変そうだな」と心配することもありましたが、客観的に見ると就活でそこまで深く抱え込んでしまう必要はないような気もします。
増田
私も正直、SNSなどで同じ大学の就活生の活動状況を見て焦ることが多々ありました。
貞木
分かります。就活でフラストレーションが溜まる人の多くは、ツイッターとかで情報を見てしまうんです。他の人の就活状況を見ると、少なからず影響を受けてしまいますからね。ただ、そこで影響を受けて気落ちするのは、すごくもったいないと思います。
大場
仕事の選び方に関しては、ある程度は自分でイメージというか準備をしておいて、就活イベントなどの機会に会社の人とすり合わせを行えば良いのかなと思います。そこで熱意を伝えられれば、良い方向につながっていくはずです。
